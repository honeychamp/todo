# Galaxy Pharmacy Management System

A premium, modern pharmacy management system built with CodeIgniter 4.

## Key Features
- **Business Intelligence**: Real-time sales and performance analytics.
- **Inventory Hub**: Manage products, categories, and vendors with shortage alerts.
- **Sales Terminal**: Interactive sales counter with doctor network integration.
- **Financial Tracking**: Automated profit/loss calculations and expense logging.
- **Security**: Secure admin authentication with profile management.

## Setup Instructions
1. Ensure PHP 8.2 or higher is installed.
2. Configure your database settings in the `.env` file.
3. Serve the application pointing to the `/public` directory.
4. Default admin credentials can be found in the system seeder.

## Built With
- CodeIgniter 4
- Bootstrap 5
- Font Awesome 6
- Chart.js
- Custom Glassmorphic CSS Engine
